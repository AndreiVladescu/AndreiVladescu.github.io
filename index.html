Security & Reverse Engineering

Hikvision DS-KV6113 Videointercom — Reverse Engineering
Chip-off firmware extraction revealing a global shared TLS private key, hardcoded backdoor shell, and undisclosed cloud uploads of facial recognition data.

Hisense 50A6N Smart TV — Firmware Analysis
Hardware teardown and flash extraction of an Android TV, identifying BleedingTooth (CVE-2020-12351) and a kernel information disclosure in the shipping firmware.
[site] [repo]
TP-Link Archer A5 — Security Analysis
Hardware teardown and firmware RE uncovering unencrypted config files, hardcoded credentials, and a DES-ECB configuration scheme that was successfully decrypted.
[site] [repo]
Exploiting the D-Link DIR-600 Router
Firmware extraction and analysis achieving unauthenticated root access by enabling the telnet daemon through exposed debug interfaces — no CVE required.
[site] [repo]
PLEM 50 C3 Laser Rangefinder — Reverse Engineering
Disassembly and protocol analysis of a budget laser rangefinder using a multimeter and logic analyzer to map internal communications for module repurposing.
